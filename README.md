## Deployment

To deploy this project run

```bash
  npm start
```

Dev Environment

```bash
  npm start:dev
```


## Run Locally

Clone the project

```bash
  git clone git@github.com:St80ene/
```

Go to the project directory

```bash
  cd zuri-task5-crud-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

  
## Tech Stack

**Server:** Node, Express
